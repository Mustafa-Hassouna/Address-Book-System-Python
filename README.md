# 📱 Address Book System (Python)

A streamlined command-line interface (CLI) application built with Python to manage personal contacts. This project demonstrates core programming logic, data organization, and user input validation.

---

## 🚀 Features
- **Add New Contacts:** Prevent duplicate phone numbers to maintain data integrity.
- **Categorization:** Classify contacts into types: `Family`, `Personal`, `Work`, or `Other`.
- **Advanced Search:** Quick lookup functionality by either **Name** or **Phone Number**.
- **Smart Deletion:** Remove specific records accurately using unique identifiers (Name/Number).
- **Error Handling:** Robust input validation using `try-except` blocks to ensure a crash-free experience.

## 🛠 Technical Concepts Applied
This project showcases the practical application of:
* **Data Structures:** Using **Lists of Dictionaries** for efficient data storage and retrieval.
* **Control Flow:** Implementation of complex `if-elif-else` logic and `while` loops for menu navigation.
* **Input Validation:** Handling `ValueErrors` to manage unexpected user behavior.
* **Algorithm Logic:** Designing custom search and filtering algorithms.
