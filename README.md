#  Address Book System
A professional Python-based CLI application for managing contacts efficiently.

##  Features
- **CRUD Operations:** Add, Search, and Delete contacts.
- **Categorization:** Classify contacts into (Family, Personal, Work, Other).
- **Data Validation:** Prevents duplicate phone numbers.
- **Error Handling:** Robust input validation using Try/Except blocks.

##  Tech Stack
- **Language:** Python 3.x
- **Data Structures:** Dictionaries & Lists for optimized data management.
